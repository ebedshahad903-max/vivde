System Prerequisites

Make sure your Windows PC has these software dependencies installed before executing the setup steps.

Windows OS Windows 10 or 11
Git CLI Version 2.40 or newer
Download Git →
Docker Desktop Latest stable release
Download Docker →
Python Environment Version 3.10 or 3.11
Download Python →
Service Status Verification Required

Docker Desktop must be running locally in your Windows taskbar before deploying or executing any container actions.

1

Choose Terminal

For Windows, the highly recommended terminal is PowerShell. Alternatively, Command Prompt (CMD) can be used.

Recommended Workflow inside VS Code:

Open VS Code, press Ctrl + ` (or go to Terminal > New Terminal), and select **PowerShell** as the shell profile.

2

Clone the Project

Run the clone command in your terminal to download the project repository from GitHub into a local directory.

powershell / cmd 
git clone https://github.com/starlite20/vividstudio.git
3

Open the Project Folder

Navigate into the newly created folder where the repository files were cloned.

powershell / cmd 
cd vividstudio
4

Create Input and Output Folders

Generate directory structures where local video inputs are loaded and generated JSON outputs are written.

powershell cmd 
mkdir input, output -Force
mkdir input output
5

Copy Sample Task File

Copy the sample task file provided in the repository into the input directory to act as the pipeline configuration payload.

powershell / cmd 
copy examples\tasks.sample.json input\tasks.json
6

Build Docker Image

Compile the application code and configure its environment locally using the project's docker configuration.

powershell / cmd 
docker build -t video-caption-agent:local .
Note on Build Time:

Building the image resolves all package requirements. This process might take 3–5 minutes the first time depending on your machine speed.

7

Run the System

Execute the Docker container, mounting the local folders into it. This reads the task configuration from `input` and places results in `output`.

powershell cmd 
docker run --rm `
  -v "${PWD}\input:/input" `
  -v "${PWD}\output:/output" `
  video-caption-agent:local
docker run --rm -v "%cd%\input:/input" -v "%cd%\output:/output" video-caption-agent:local
8

View the Output

Read the generated contents in `results.json` to verify that the agent execution finished successfully.

powershell cmd 
Get-Content .\output\results.json
type output\results.json

9

Run the Frontend (Optional)

Start the local Python server backend to serve the application user interface locally in your browser.

powershell / cmd 
PYTHONPATH=src python -m video_caption_agent.server
Access URL:

Open your browser and navigate to http://localhost:8000 to open the interface. (You can also open the file `frontend/index.html` directly in a browser to use the client sandbox).

Open Localhost:8000

Simple Summary

The fastest path to get up and running for a complete beginner:

  • 1 Open VS Code and launch terminal in PowerShell mode.
  • 2 Clone repository: git clone https://github.com/starlite20/vividstudio.git
  • 3 Navigate into directory: cd vividstudio.
  • 4 Run the setup commands in sequence to build and execute the docker stack.
  • 5 Verify final outputs generated at output/results.json.
